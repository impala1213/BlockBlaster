
==========================================
======= 하나하나 만들기 귀찮아서 만든 더 귀찮은 ========
================= 블럭메이커 =================
==========================================


- 사용 위치
Assets/Z_Other_LSM/00Scenes 폴더 내에 있는
 "BlockMakerPreviewScene" 에서 가능합니다.

- 사용 방법
 - Hierarchy 에 있는 'ShapeMaker' 오브젝트를 클릭한다.
 - Inspector창에 있는 'ShapeMaker' 스크립트를 확장시킨다.
 - '데이터 선택'의 드롭다운을 확장하여 변경할 파일 혹은 새로 생성을 누른다.
 - 이후 원하는대로 사용한다.

- 해당 파일의 저장 위치
  - Assets/PieceDefinition 폴더에 전부 저장하며, 해당 폴더에 있는 PieceDefinition의 파일들을 모두 받아옵니다.

- 기능
  - 드래그를 사용하여 만들 수 있는 아마도 간편한 기능.
  - 회전 및 반전 기능
  - 파일 변경 기능.
  

- 없는 기능
  - 파일 삭제 기능 (만들려다가..)


- 주의 사항
  - 파일의 이름을 지정할 수 있습니다. 해당 파일의 이름이 바뀐다면 다른 파일로 인식이 됩니다.
    - 즉 원래 만들었던 파일의 '파일 이름'만 변경하려고 한다면 해당 폴더에 가서 직접 삭제하셔야합니다.
  - 파일이름이 같다면 덮어쓰기가 활성화됩니다. 버튼을 한번 더 누른다면 저장이 되니 한번 더 눌러주세요.
