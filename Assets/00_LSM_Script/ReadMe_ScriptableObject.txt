
모든 ScriptableObject는
Create / ScriptableObject / LSM 에 존재합니다.

***
ScriptableObject는 반드시 밑의 폴더에 넣어주셔야합니다.
Assets/Resources/Scriptable/LSM/ (폴더)
만들기만 하면 자동으로 불러오게 지정하였습니다.
***

=================================================
==============     업적 (SO_Achievement)
=================================================

업적의 이름과 타입 등에 대해서 설정합니다.

----- 이름
_name (string)
  - 업적의 이름을 지정합니다.

----- 타입
_type (LSM.E_Achievements_Code)
  - 업적의 타입에 대해 반환합니다.
  *** Enum 타입이기에 업적을 추가할 경우 스크립트를 변경해야하니 작성자에게 말해주세요! ***

----- 레벨 변화에 필요한 값
_invert (int)
  - _invert의 값에 따라서 레벨이 변화합니다. 현재 스코어를 _invert로 나누어 레벨을 확인합니다.
  - 레벨은 '등차수열'로 증가합니다.




=================================================
==============     음악 (SO_SoundClip)
=================================================

노래에 대하여 타입과 코드를 설정합니다.

----- 노래 타입
_clipType (E_SoundClip_Type)
  - BGM 혹은 SFX로 지정할 수 있습니다.

----- 노래 코드
_code (string)
  - 노래의 이름을 변경하는 것은 애매하다고 느껴서 '코드'를 지정할 수 있게 만들었습니다.

----- 노래 파일
_clip (AudioClip)
  - 노래를 넣으시면 됩니다.
